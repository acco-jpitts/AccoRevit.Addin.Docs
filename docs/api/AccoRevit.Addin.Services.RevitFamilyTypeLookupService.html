Remarks

Results are cached per file path to avoid repeatedly opening the same family file, which can be expensive.

This service opens the family document using Autodesk.Revit.ApplicationServices.Application.OpenDocumentFile(System.String) and closes it in a finally block. The lookup is read-only and does not modify any documents.

If the file is missing, invalid, not a family document, or if any exception occurs while opening/reading/closing, this service returns an empty list and caches an empty result for the given path.

public IReadOnlyList<string> GetFamilyTypeNames(string familyFilePath)

Parameters

familyFilePath string

Full path to a Revit family file (.rfa). If the path is null/empty/whitespace or the file does not exist, an empty list is returned.

Returns

IReadOnlyList<string>

A read-only list of distinct family type names (symbol names) sorted alphabetically (case-insensitive). Returns an empty list if the file cannot be opened, is not a family document, or contains no symbols.

Remarks

The result is cached per file path. Subsequent calls with the same familyFilePath will return the cached list without opening the family document again.
