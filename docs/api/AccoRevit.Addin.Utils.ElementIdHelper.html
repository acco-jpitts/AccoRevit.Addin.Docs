Remarks

Revit 2024+ uses 64-bit element ids (Autodesk.Revit.DB.ElementId.Value), while earlier versions use 32-bit ids (Autodesk.Revit.DB.ElementId.Value). This helper centralizes the conditional compilation required to convert and compare ids safely across supported targets.
